## 简易的 PHP - WING

# 目录结构
```
├── app
│   └── config
│       ├── extra          其它配置文件
│       ├── config.php     基础配置
│       ├── database.php   数据库配置
│       └── common.php app 公共方法
├── public          网站首页
│   ├── index.php
│   └──static
├── README.md
├── runtime
│   ├── cache 模板缓存
│   └── log   日志
└── phpwing
    ├── config
    │   ├── config.php     基础配置
    │   ├── common.php     框架公共方法
    │   └── global.php     通用配置
    └── lib  
        ├── library     其它内核类
        │   ├── composer
        │   │   └── vendor
        │   ├── composer.json
        │   ├── composer.lock
        │
```
